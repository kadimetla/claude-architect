# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it responsibly:

- Email: [tim@techtrainertim.com](mailto:tim@techtrainertim.com)
- Do NOT open a public issue for security vulnerabilities

## Scope

This repository contains educational materials and example code. The code examples are intended for learning purposes and should not be deployed to production without proper security review.

## Sensitive Content

This repository must never contain:

- API keys, tokens, or credentials
- The CCA-F Exam Guide PDF (Anthropic Confidential)
- Actual exam questions or answers
- Content from the `private/` directory
